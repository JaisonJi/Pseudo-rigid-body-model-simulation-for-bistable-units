# Pseudo-rigid-body-model-simulation-for-bistable-units
The codes are used to generate the simulation results in Figure 3, Figure 4 and Figure 5.
Pseudo-Rigid-Body Model (PRBM) returns key descriptors of the force-displacement response of bistable units inherited in multi-stable tensegrity metamaterials, including peak/valley forces and their displacement locations, effective stiffness, and the response of parallel assemblies. The PRBM is constructed by Lagrangian mechanics and are simulated under MATLAB environment.
Costumized parameters shall be defined in 'Parameters definition' section in PRBM_MAIN.mlx. Including: 
Geomatric paramters (l1, L, r1, H, l3, r2, phi, b, h)
Material property (E, Iz, des)
Modelling design parameters (lmd1, lmd3)
Loading condition (v, g)
Mofifier factor (fct1, fct2, fct3) (if needed)
